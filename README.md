# BarcodeMenuBar
## 项目简介
macOS 菜单栏条码工具，可从剪贴板文本生成条码或二维码。

### 项目主要功能
- 读取剪贴板中的文本并生成对应图像。
- 在 Code 128 条码和二维码之间切换。
- 提供 S/M/L 尺寸预设，调整预览和导出大小。
- 支持自定义全局快捷键唤起和更新快捷键。
- 将生成的条码图片直接复制回剪贴板。

# InvestView
## 项目简介
个人投资看板，基于 Next.js App Router、TypeScript 和 Tailwind CSS 构建。

### 项目主要功能
- 实时或近实时获取指数和市场数据，优先展示真实接口返回结果。
- 展示核心估值指标，包括价格、PE-TTM、多周期涨跌和 3/5/10 年分位。
- 通过颜色分区提示估值状态，区分买入区、观察区和警戒区。
- 提供历史图表，展示 PE 趋势及关键阈值线。
- 支持真实 API 失败时自动降级为 Mock 数据，保证页面可用性。
